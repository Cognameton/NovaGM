The NovaGM project aims to be a cross-platform (Windows/Linux) desktop Game Master tool and multiplayer host for TTRPGs. It utilizes C# with Avalonia UI and an embedded ASP.NET Core server for Jackbox-style browser client joining over local Wi-Fi.

Key requirements fulfilled during the last trajectory include:
1. LLM Automation: Full support for local LLamaSharp inference for Controller, Narrator, and Memory roles.
2. Character Generation: Implementation of quick auto-generated character shortcuts (Random, Fighter, Rogue, Mage, Pilot, Engineer, Scientist) on both desktop and browser client.
3. Mission System: Creation of a complete feature for saving current sessions as standalone missions in a lore journal, including save/load dialogs.
4. Multiplayer/UI: Implementation of player tracking display, real-time narration streaming fixes, and enhanced context-aware AI responses.
5. Extensibility: The architecture must support future DLC/lore packs for classes, races, and story hooks, which is the current focus of development.
